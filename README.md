# Clausr-main

Clausr is a contract lifecycle and renewal reminder web app built with Next.js and Supabase.

## Run locally

1. Install dependencies:
   - `npm ci`
2. Copy env template:
   - `cp .env.local.example .env.local`
3. Fill values in `.env.local`
4. Start app:
   - `npm run dev`

## Deploy

Deploy on Vercel and configure Supabase URLs + env vars as documented in `SETUP.md`.
